# QUANTUM TOOLS

A futuristic, modern, and responsive landing page with HTML, CSS, and JavaScript.

## Features

- Modern dark theme with glowing accents and light mode toggle
- Responsive design for all device sizes
- Interactive particle.js background
- Glassmorphism UI with advanced animations
- Smooth scrolling navigation
- Newsletter subscription form
- Accessibility enhancements

## Technologies Used

- HTML5 with semantic markup
- CSS3 with custom variables and animations
- JavaScript for interactive elements
- Particles.js for background animations
- Font Awesome for icons

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers

## Accessibility Features

- Keyboard navigation support
- Skip to content link
- ARIA attributes
- Reduced motion support
- Color contrast enhancements

## File Structure

- `index.html` - Main HTML structure
- `style.css` - Core styling
- `animations.css` - Animation effects
- `preloader.css` - Preloader styling
- `enhancements.css` - Additional UI improvements
- `enhanced-script.js` - JavaScript functionality

## Future Enhancements

- Performance optimizations for animations
- Additional tool detail pages
- User account system
- Dark/light mode preference persistence
